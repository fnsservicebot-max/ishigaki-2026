# 2026 盛夏沖繩前哨站・石垣島 5 日瘋 🌺

石垣島 5 天 4 夜跳島之旅網頁版（GitHub Pages）

## 📅 行程

- **Day 1 (6/19)**：川平藍與松鼠猴的西海岸巡禮
- **Day 2 (6/20)**：夢幻沙洲（跳島日）
- **Day 3 (6/21)**：竹富島水牛車（跳島日）
- **Day 4 (6/22)**：深入地底奇景 + 採購
- **Day 5 (6/23)**：09:25 班機起飛返台

## 🔗 線上版

https://fnsservicebot-max.github.io/ishigaki-2026/

## 🛠️ 技術

- 純 HTML + CSS（無框架）
- Tailwind 不使用，全部 inline CSS
- PWA manifest（可加入 iPhone/Android 主畫面）
- localStorage 儲存 checklist 狀態

## ✨ 功能

- 5 天時間軸切換（tab 按鈕）
- 每個景點的 Google Maps 導航連結
- 餐廳卡片
- 飯店資訊
- 待辦清單（自動記憶打勾狀態）
- 預算估算表
- 行程提醒（松鼠猴互動注意事項、採購秘訣）
